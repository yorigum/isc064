Security Level Tidak Memadai

Kemungkinan Terjadi Karena :

  • Proses konfigurasi security level belum dijalankan untuk program-program tambahan yang baru saja dipasang.
  • Akses security anda tidak dapat membuka halaman / modul / proses / data yang bersangkutan. Silakan menghubungi divisi IT Security untuk otorisasi dan keterangan lebih lanjut.
  • Data yang akan dibuka, memiliki akses Project atau Perusahaan yang tidak Anda miliki.
